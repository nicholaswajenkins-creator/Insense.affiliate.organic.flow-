01
Campaign flow — where affiliate branches
Overview of the 4-phase journey with affiliate touchpoints highlighted
🔑
Core placement decision: Organic Affiliate is a third campaign type — not a separate module or nav item. The existing 4-step campaign creation flow is preserved exactly. Three screens get new fields injected. One entirely new concept (affiliate link auto-generation) appears inside the existing creator invite step.

New — Commission rate column
Shows the rate set in brief builder. Could be made per-creator editable here if brand wants to offer different rates to different creators (e.g. higher rate for larger accounts).

Unique link per creator, auto-generated by Insense (format: insense.pro/a/{creatorcode}{SKU}). Read-only for the brand. Automatically embedded in the invite email. Creator uses it to tag products.

Field spec — all new and extended UI elements
Complete reference for engineering handoff
ScreenField / ElementTypeSource / LogicStatus
Campaign type campaign_type = "organic_affiliate" Enum selection New value added to existing campaign_type enum. Conditionally reveals affiliate setup fields. NEW
Campaign type commission_rate Number input (%) Brand-set. Default: 10%. Stored on campaign object. Propagated to all creator deals. NEW
Campaign type attribution_window_days Select (integer) Default: 30 (YouTube) or 7 (Instagram) based on platform. Brand can override. NEW
Campaign type platform_commerce_connection OAuth flow Google Merchant Center (YouTube) or Meta Commerce Manager (Instagram). Stored as token on brand account. NEW
Brief builder selected_sku_ids[] Multi-select picker Product catalogue pulled from GMC or Commerce Manager API. Array of SKU IDs stored on campaign brief. EXTENDED
Brief builder base_fee_per_creator Number input ($) Optional fixed fee in addition to commission. Default: $0. Stored on campaign brief. NEW
Brief builder Commission + attribution display Read-only display Displays values set in campaign type step. Editable here to allow per-brief override. EXTENDED
Creator invite affiliate_link Auto-generated string Generated by Insense on deal creation. Format: insense.pro/a/{creator_code}{sku_code}. Unique per creator per campaign. Read-only for brand. NEW
Creator invite Commission column in deal table Display (from campaign) Shows commission_rate from campaign. Could be made per-creator editable in v2. EXTENDED
Creator invite Invite email payload Email template update Existing invite email extended to include affiliate_link and commission_rate. No new email template needed — inject into existing template. EXTENDED
Performance gmv KPI card Computed (from API) YouTube: from GMC Reporting API (youtube_creator_performance_view). Instagram: from CAPI purchase events. Aggregated across all creators. NEW
Performance orders KPI card Computed (from API) Count of confirmed purchase events attributed to campaign creators within attribution window. NEW
Performance commission_owed KPI card Computed gmv × commission_rate. Updated daily. Triggers wallet credit to creator on confirmed sale. NEW
Performance GMV / Orders / Commission / Source columns Table column additions Four columns added to existing creator performance table. Per-creator values from GMC API (YouTube) or CAPI (Instagram). Data source shown for transparency. EXTENDED
Performance re_invite_cta button Conditional button Appears when creator's gmv ≥ campaign re_invite_threshold (set in campaign setup, default: $500). Opens long-term collab invite flow. NEW
All screens Sidebar "Organic Affiliate" nav item Filter/nav item New item in Campaigns sidebar to filter to affiliate campaigns only. Existing nav structure unchanged. NEW
